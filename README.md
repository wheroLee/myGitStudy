## myGitStudy

### 현재 상태 확인
> git status

### git 로그 확인
> git log

### git 저장소 생성하기
> git init

### git 저장소 복제 및 다운로드
> git clone https:~~~

### 저장소에 파일 추가
> git add
> git add .
> git add *

### 커빗 생성
> git commit -m "message"

### 변경 사항 원격 서버 업로드 (push)
> git push origin master

### 원격 저장소 변경 내용을 로컬 저장소로 가져와
> git pull

### 변경 내용 비교 
> git diff branchA branchB
